1. What is the difference between var, let, and const?

#var is function-scoped and can be redeclared or reassigned.

#let is block-scoped, can be reassigned but cannot be redeclared in the same scope.

#const is block-scoped, cannot be reassigned or redeclared, used for constants.

2. What is the spread operator (...)?

#The spread operator allows an iterable like array or object to be expanded in places where zero or more elements/arguments are expected.

3. What is the difference between map(), filter(), and forEach()?

#map() → transforms each element and returns a new array.
#filter() → selects elements based on a condition and returns a new array.
#forEach() → executes a function on each element, returns nothing.

4. What is an arrow function?

#Arrow Function A compact way to write functions in JavaScript. It is a ES6 version of writing function.

5. What are template literals?

#Template Literals Strings that allow embedded expressions using backticks.Supports multi-line strings inside JS.
